## Ключевые навыки

### Bash-скрипты
- Переменные, условные операторы (if-else), циклы (for)
- Функции и аргументы командной строки ($1, $2)
- Работа с файлами: создание, перемещение, архивирование (tar)
- Проверка доступности сайтов (ping), анализ использования диска (df)

### Автоматизация (crontab)
- Создание и редактирование cron-заданий
- Ежедневное резервное копирование директорий
- Мониторинг дискового пространства с пороговыми значениями
- Очистка временных файлов старше N дней
- Задания по расписанию: будние дни, логирование выполнения

### Системное программирование (C++)
- Создание и синхронизация процессов: fork(), waitpid()
- Семафоры System V: semget(), semctl(), semop()
- Блокировка родительского процесса до завершения дочернего
- Корректное освобождение IPC-объектов
